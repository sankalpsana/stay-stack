# StayStack: The PG Operating System 🏢

StayStack is a comprehensive, end-to-end management platform designed to streamline operations for PG (Paying Guest) and co-living spaces. 

In tech hubs where PGs are often managed via chaotic WhatsApp groups and Excel sheets, StayStack brings order. It replaces manual rent collection, unorganized maintenance requests, and high food waste with a clean, automated, and centralized system.

Built with a microservices architecture utilizing Spring Boot and Python, and designed for scalable deployment on Kubernetes, StayStack serves the needs of both property owners and their tenants.

## 🚀 Core Modules

* **Automated Financials & Billing:** Automated monthly rent generation, prorated calculations, security deposit tracking, and integrated payment flows.
* **Occupancy & Lifecycle Management:** Real-time visual dashboard of bed availability, digital KYC onboarding, and automated move-out notice tracking.
* **Operations Helpdesk:** A centralized ticketing system for tenants to raise maintenance requests (plumbing, electrical, Wi-Fi) with real-time status tracking and SLA escalations.
* **Smart Mess Management:** Kitchen dashboard featuring daily menu publishing and a one-tap meal "opt-out" feature for tenants, providing exact headcounts to drastically reduce food waste.

## 🛠️ Architecture & Tech Stack
* **Backend:** Microservices architecture (Spring Boot / Python)
* **Infrastructure:** Containerized and orchestrated via Kubernetes
* **Version Control:** Git
